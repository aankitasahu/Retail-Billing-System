# Retail Billing System

## Project Description
A full-stack web application to manage retail products. Users can add products and view them in real-time.

## Features
- Add product (name, price, quantity)
- View all products
- Real-time update using REST API

## Tech Stack
- Frontend: React
- Backend: Spring Boot (Java)
- Database: MySQL
- Tools: Eclipse, VS Code, Postman

## Project Structure
- backend → Spring Boot application
- frontend → React application

## How to Run

### Backend
1. Open in Eclipse
2. Run RetailBillingApplication.java

### Frontend
1. Go to frontend folder
2. Run:
   npm install
   npm start
